### Простой пример
```
int func(int x);

int main()
{
    cout << func(5);
}
```
Возникнет ошибка компиляции из-за ошибки линкера - функция объявлена, но не определена.

После определения функции всё заработает:
```
int func(int x) 
{
    return x+=x;   
}

int main()
{
    cout << func(5);
}
```
